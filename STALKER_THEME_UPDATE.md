# 🎮 Zone Stalker - STALKER Theme Update

## 📋 What's New

The Sky Islands mod has been completely revamped with a **STALKER theme** while preserving all original gameplay mechanics. The mod is now called **Zone Stalker** and provides an authentic Chernobyl Exclusion Zone experience.

## 🔄 Major Changes

### 🏷️ **New Identity**
- **Mod Name**: ~~Sky Island~~ → **Zone Stalker**
- **Scenario**: ~~Warper~~ → **Zone Stalker** 
- **Setting**: Underground laboratory in the Zone (instead of floating island)

### 🎒 **Items Renamed**
| Original | STALKER Theme |
|----------|---------------|
| Quickheal pill | Medkit injector |
| Warp shards | Artifact fragments |
| Warped hauler's harness | Artifact gravity harness |
| Earthbound pill | Anomaly stabilizer |
| Homeward mote | Emergency beacon |
| Warped animal carrier | Anomalous specimen cage |

### 🩺 **Effects Updated**
| Original | STALKER Theme |
|----------|---------------|
| Earthside | In the Zone |
| Warp Sickness | Anomaly Sickness |
| Warp Disintegration | Anomaly Breakdown |
| Warpcloaked | Phase-shifted |

### 🌍 **Lore Changes**
- **Base**: Secret underground laboratory with spatial anomaly technology
- **Expeditions**: Teleportation to random Zone locations for artifact hunting
- **Technology**: Scientific anomaly research instead of mystical warp powers
- **Missions**: Faction contracts and Zone operations
- **Time Limits**: Anomaly exposure effects instead of "warp sickness"

## ✅ **What's Preserved**

- ✅ Base building and progression

## 📥 **Installation**

1. Download the updated mod files from this repository
2. Replace your existing Sky Islands mod folder
3. In CDDA, look for **"Zone Stalker"** instead of "Sky Island"
4. Create a custom character (Play Now still won't work)
5. Start your Zone operations!
